# Basic Example

This example demonstrates basic usage of the module.

## Usage

```bash
# Initialize Terraform
terraform init

# Plan the deployment
terraform plan

# Apply the configuration
terraform apply

# Clean up when done
terraform destroy
```

## Requirements

| Name | Version |
|------|---------|
| terraform | >= 1.0.0 |

## Inputs

No inputs.
